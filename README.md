# 个人摄影集展示网站

一个现代化的个人摄影作品展示网站，支持按时间和城市分类展示照片，包含完整的后台管理功能。

## ✨ 功能特点

### 前台展示
- 🎨 **现代化设计**：简约大气的卡片式布局，支持复古文艺风格
- 📱 **响应式设计**：完美适配桌面端、平板和手机
- 🖼️ **照片展示**：支持照片放大查看，键盘导航
- 🏷️ **分类展示**：按时间+城市组织作品集（如"2023上海"、"2025布达佩斯"）
- 📝 **图文结合**：每个作品集支持丰富的文字描述
- ⚡ **性能优化**：图片懒加载，流畅的动画效果

### 后台管理
- 🔐 **安全登录**：密码保护的管理后台
- ➕ **作品管理**：创建、编辑、删除作品集
- 📸 **照片管理**：批量上传和管理照片
- 💾 **数据持久化**：本地存储，支持数据导入导出
- 📊 **统计信息**：作品数量、访问统计等

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装依赖
\`\`\`bash
npm install
\`\`\`

### 启动开发服务器
\`\`\`bash
npm start
\`\`\`

访问 http://localhost:3000 查看网站

### 构建生产版本
\`\`\`bash
npm run build
\`\`\`

## 🎯 使用指南

### 后台管理
1. 访问 `/admin` 进入管理后台
2. 默认密码：`admin123`
3. 创建新的作品集，填写标题、地点、年份等信息
4. 上传照片并添加描述
5. 发布后即可在前台查看

### 自定义配置
- 修改 `src/data/mockData.ts` 中的示例数据
- 在 `src/pages/About.tsx` 中更新个人信息
- 调整 `src/index.css` 中的主题色彩

## 🎨 设计特色

### 视觉风格
- **简约现代**：干净的布局，充足的留白
- **复古元素**：Playfair Display 字体，温暖的色调
- **渐变色彩**：紫色渐变主题，营造艺术氛围

### 交互体验
- **流畅动画**：hover 效果，页面切换动画
- **直观操作**：清晰的导航，易用的管理界面
- **快捷键支持**：图片浏览支持键盘操作

## 📱 响应式设计

- **桌面端**：1200px+ 三列布局，大图展示
- **平板端**：768-1200px 两列布局，适中尺寸
- **手机端**：768px- 单列布局，触摸优化

## 🚀 部署选项

### 1. Vercel（推荐）
\`\`\`bash
npm install -g vercel
vercel
\`\`\`

### 2. Netlify
1. 构建项目：`npm run build`
2. 将 `build` 文件夹拖拽到 Netlify
3. 配置自定义域名

### 3. 传统服务器
1. 构建项目：`npm run build`
2. 将 `build` 文件夹上传到服务器
3. 配置 Nginx 或 Apache

## 🔧 技术栈

- **样式方案**：CSS Modules + 原生 CSS
